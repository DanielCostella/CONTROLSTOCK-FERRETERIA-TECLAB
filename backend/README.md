# Backend - Control de Stock Ferretería

## Estructura inicial

- `src/index.js`: Servidor Express y endpoint de prueba.
- `src/db.js`: Conexión a MySQL usando variables de entorno.
- `.env`: Configuración de conexión a la base de datos.

## Comandos útiles

- Instalar dependencias: `npm install express mysql2 dotenv`
- Iniciar servidor: `npm start`

## Endpoint de prueba

GET `/api/test-db` → Prueba la conexión a la base de datos.
